std::vector< Point > computeHull ( std::vector< Point > & points)

Вычисляет выпуклую оболочку множества точек

Функция для вычисления выпуклой оболочки

Реализует модифицированный алгоритм Грэхема. Использует сортировку точек и последовательное построение нижней и верхней частей выпуклой оболочки.

Parameters
pointsВектор точек, из которых строится выпуклая оболочка
Returns
std::vector<Point> Точки выпуклой оболочки в порядке обхода
Exceptions
std::runtime_errorЕсли количество точек меньше трёх

Используется для проверки, находятся ли точки на одной линии, поворот происходит по часовой стрелке или против часовой стрелки.

  • 0, если точки коллинеарны
  • 1, если поворот по часовой стрелке
  • -1, если поворот против часовой стрелки
